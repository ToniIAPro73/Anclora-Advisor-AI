# Anclora Advisor AI

Aplicación de asesoría fiscal, laboral y mercado inmobiliario para autónomos pluriactividad.

## Stack

- Next.js 15 + React 19
- TypeScript
- Supabase PostgreSQL + pgvector
- Anthropic Claude + Ollama Mistral
