# Product Spec v0 - Anclora Advisor AI

## Producto
Asistente conversacional para consultas fiscales, laborales e inmobiliarias con backend Next.js y soporte de datos en Supabase.

## Objetivo
Entregar respuestas utiles, seguras y trazables para usuarios finales.

## KPIs iniciales
- Tiempo de respuesta estable.
- Tasa de errores API controlada.
- Calidad percibida de respuesta del chat.
