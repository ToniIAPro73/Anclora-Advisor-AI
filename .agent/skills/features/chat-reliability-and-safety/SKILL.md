# SKILL: Chat Reliability and Safety

## Cuando usar

Cambios en prompt orchestration, contrato de `/api/chat`, validaciones, limites y seguridad.

## Checklist

1. Contrato de request/response estable.
2. Manejo de errores tipado y consistente.
3. Timeout y reintentos controlados.
4. Filtros basicos de entrada maliciosa.
5. Sin fuga de datos internos en respuestas.
