# SKILL: Conversation UX and Accessibility

## Cuando usar

Cambios de interfaz del chat, estados de carga/error, entrada de usuario y navegacion.

## Checklist

1. Estados claros: idle/loading/success/error.
2. Mensajes de error accionables.
3. Navegacion con teclado y etiquetas accesibles.
4. Sin regresiones visuales en mobile/desktop.
